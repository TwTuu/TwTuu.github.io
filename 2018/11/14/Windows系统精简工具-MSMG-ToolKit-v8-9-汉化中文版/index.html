Windows系统精简工具 MSMG ToolKit v8.9 汉化中文版

发表于 2018-11-14 | 分类于 Windows资源 |

MSMG Toolkit 是一款很强大的微软系统精简工具。强大到超乎你的想象，可以精简Windows10系统很多组件，而且还不是跟其余的精简工具一样是暴力删除精简，这个是调用了微软系统自带的 Dism 命令来精简，相当于是Dism命令图形化，使用MSMG Toolkit可以完美的移除Windows Apps组件、多媒体组件、本地系统语言文件、硬件支持、系统组件、移除远程控制、移除驱动程序等等，特别还支永久移除掉讨厌的Edge浏览器、Cortana微软小娜、支持移除Windows Defener 组件。

资源简介

支持Windows 10红石4源码操作系统，Windows 10 Redstone 4 WADK (v10.0.17133.1)工具，更新的Windows 10红石4源操作系统包，工具包Delta更新包，适用于那些只想下载更新的文件而不是整个包的人，语言包(Windows 10红石4)对于MSMG工具包来说是不完整的，因为下面的WinPE包是不可用的，因此，集成语言包用于安装映像工作，但对于引导/恢复映像，需要上述文件，一旦上述文件可用，包将被更新。转换->转换版本已更新为Windows 10红石4，其余版本将在下次更新中更新，集成微软默认的Metro应用程序功能将在应用程序可用后更新，删除Windows组件:防御者、PeopleExperinceHost和其他使用工具kithelper方法的组件将在下一个版本中更新。

MSMG ToolKit 使用教程

MSMG ToolKit 系统要求：

  • Windows 7 with SP1 / Windows 8.1 / Windows 10 / Server 2008 R2 / Server 2012 R2 / Server 2016 安装光盘 DVD/ISO。
  • 用于维护 Windows 7 / Windows 8.1 源映像的 Windows 7 with SP1 / Windows 8.1 / Windows 10 主机操作系统。
  • 用于维护 Windows 7 / 8.1 / 10 源映像的 Windows 8.1 / Windows 10 主机操作系统。
  • 组件 ToolKitHelper.exe 需要 Microsoft .NET Framework 4.7.1。

MSMG ToolKit 已知问题

  • 在启用防病毒程序时使用工具箱会影响工具箱的操作。
  • 由于 DISM 的工作方式，使用 Windows 10 刷新版映像时，工具箱有可能会无法正常工作。
  • 工具箱中的组件 ToolKitHelper.exe 可能被标记为病毒/木马/恶意软件程序，请不要担心，这只是误报。
  • 本工具箱不能/不支持使用其他类似工具（如 NTLite / WinToolkit / WinReducer）对工具箱维护的源映像进行后期维护。
  • 在删除组件后集成/安装 Windows 更新/累积更新很少能够恢复已删除的组件资源或快捷方式文件/文件夹。
  • Windows 8.1 – 集成 Windows 远程服务器管理工具（RSAT）时将会单独集成，但是当它在集成所有其他组件之后进行集成时它将会出现错误代码 0x80092004 中断集成。
  • Windows 10 Threshold 1 – 由于缺少服务堆栈的可用性，因此无法集成最新的累积更新。
  • Windows 10 Threshold 1 – 删除除商店和所有 Windows 组件之外的所有应用将破坏商店应用和设置 -> 应用程序 -> 商店还原功能。
  • Windows 10 Redstone 2 – 在安装 OOBE 期间，在网络选择屏幕上，当你选择跳过时，安装程序将显示错误 – “出现问题，你可以再试一次，或者暂时跳过此步骤”，只需选择跳过。
  • Windows 10 Redstone 2/3/4 – 选择 Win32 计算器中的单位转换菜单会引发内存引用错误。
  • Windows 10 Redstone 2/3/4 – 当自定义用户帐户图片被集成时，在登录屏幕中将不显示图片，虽然它已在开始菜单用户图标中显示。
  • Windows 10 Redstone 3/4 – 在 Microsoft 遥测组件下删除统一遥测客户端（Asimov）会破坏 Sticky Notes 应用和通过 Windows 更新安装驱动程序的功能。
  • Windows 10 Redstone 4 – 从具有预先集成的累积更新的映像中删除 Windows Media Player 组件将破坏 Windows 安装。
  • Windows 10 Redstone 3/4 – 据报告，在格式化 USB 功能中，Diskpart 命令“list”在脚本中使用时不起作用。
  • Windows 10 Redstone 3/4 – 据报告，集成 Win32 计算器包会破坏 Office 365 预览版处理程序。

MSMG ToolKit 一般用法

  • 将 Windows 源 ISO/DVD 映像/光盘提取内容/复制到工具箱的 文件夹。
  • 在 Start.cmd 上双击并选择是以管理员身份运行。
  • 单击工具箱的命令窗口控制框，然后选择属性。
  • 如果使用 Windows 7 / Windows 8.1 主机操作系统，则转到字体选项卡并将字体设置为 Consolas，字体大小设置为 16。
  • 转到布局选项卡并在屏幕缓冲区大小组框中将高度增加到 1000。
  • 按“A”键同意工具箱的最终用户许可协议（EULA）。
  • 按 Enter 键以继续执行。
